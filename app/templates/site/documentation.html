How it works

Client Experience

  • GazeIntent calibrates your eyes by running through a test environment with 9 points. Look at each visual cuesto capture your gaze.
  • Make sure you are sitting in a comfortable posture.
  • Face the front of the eye tracker and screen. Your eyes should be in the center of the track box.
  • Adjust yourself to be at the optimal distance to the eye tracker.

    • Admin Experience

  • As an authenticated admin user, GazeIntent collects and displays all the clients generated data.
  • Test Results table shows the record id, user id, start timestamps, end timestamps, total time, type and the record create time.
  • Heatmap visualization shows the total seconds attention was given to any given cell for each test session.
  • The user table shows the profile information for each user registered on GazeIntent.
  • Calibration Results table shows stats of the calibration sessions from the registered test users.
